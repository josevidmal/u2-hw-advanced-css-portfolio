Weather Dashboard

In this application the user types the name of a city in the input field and clicks the Search button. If the input does not match a valid city name, an alert window is shown, asking the user to choose a valid name. Then, using the OpenWeather Server-Side APIs the application retrieves current weather and 5 day weather forecast information for the city entered by the user. After that, the application selects specific information (city name, weather icon, temperature, wind speed, humidity and UVI) to display in the current weather and forecast sections.

  • GitHub Repo
  • Technologies: JavaScript, HTML, CSS, Moment.js, Web APIs, Server-Side APIs, Google Fonts
